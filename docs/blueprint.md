# **App Name**: King's Court

## Core Features:

- AI Chat Interface: Provide a user-friendly chat interface for interacting with the AI chatbot.
- General Knowledge Base: Load the AI chatbot with a comprehensive knowledge base to answer a wide array of questions.
- Memory Tool: Implement a memory tool that allows the chatbot to remember past conversations and user preferences.
- Persistent User Profiles: Create and store user profiles to personalize the chatbot's responses and memory.
- Contextual Understanding: Enable the chatbot to understand and respond appropriately to the context of the conversation.
- Custom AI persona: King A.J.: Define and maintain a specific personality, tone, and communication style for the AI chatbot (King A.J.).
- Admin Interface: Develop an admin interface to upload and modify the AI's knowledge base and manage user profiles.

## Style Guidelines:

- Primary color: Deep royal blue (#4169E1) for authority and wisdom.
- Background color: Light grayish-blue (#D8E2EF) for a clean and calm interface.
- Accent color: Gold (#FFD700) for highlights and important information.
- Font pairing: 'Playfair' for headings (serif), and 'PT Sans' for body (sans-serif) to provide elegance and readability.
- Use clean, minimalist icons to represent different functions and categories within the app.
- Maintain a simple, intuitive layout with clear sections for chat, knowledge base access, and user settings.
- Incorporate subtle animations to enhance user experience and provide feedback on interactions.
